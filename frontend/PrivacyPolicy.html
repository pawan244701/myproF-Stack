Privacy Policy

Last Updated: July 2026

This platform operates as an open-source development portfolio and engineering sandbox. In the spirit of total transparency, the entire codebase—including all routing logic, authentication flows, UI components, and game modules—is completely public and accessible on GitHub. The only layer kept confidential is the private database credentials to protect the platform's active environments.

Data Handling & Authentication

While the application code is entirely public, your registration data is handled securely. The system collects email addresses, Real Names, Date of Births, Gender (Optional), Country, States, Districts, an Unique Username, and a Password solely to manage account sessions and process OTP verification. All user passwords are encrypted using server-side hashing logic before being stored in our database, ensuring your raw credentials are never visible.

CRITICAL SECURITY NOTE

Because this platform functions as a public development sandbox, visitors are highly advised NOT to use passwords that match their real-world personal accounts (like official Google, banking, or social media profiles). Please register using sandbox or test-specific credentials only.

Account Deletion

If you have created a test account and wish to have your database records completely purged from the system backend storage tables, please contact me directly using the communication links provided in the main page.
