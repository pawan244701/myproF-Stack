# User Registration & Authentication Flow 🔐

This was my very first full-stack learning project! It features a complete user registration and login system, integrated with email authentication and a database backend.

🚀 **Live Demo:** [Click here to view the deployed app on Render](https://pawan244701.onrender.com/login.html) Or visit: (https://pawan244701.onrender.com)

---

## 🛠️ Tech Stack Used

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Hosting:** Render

---

## 🌟 Key Features Developed

* **User Registration:** Secure collection of user details during signup.
* **Email Authentication:** Verification flow to ensure users sign up with valid email addresses.
* **Login System:** Session handling to verify returning users.
* **Database Integration:** Connecting Node.js to a MySQL database to store and retrieve user credentials.
